# Monitor Workflow Configuration

## Required Secret

Add repository secret for health monitoring:

```
HEALTHZ_URL = https://<your-domain>/healthz
```

## Setup Instructions

1. Go to GitHub repository Settings
2. Navigate to Secrets and variables â†’ Actions
3. Add new repository secret:
   - Name: `HEALTHZ_URL`
   - Value: Your production health check endpoint
4. Re-run monitor workflow: `.github/workflows/monitor-arena.yml`

## Expected Health Response

The monitor expects JSON response:
```json
{
  "ok": true,
  "version": "2.0.0", 
  "arena": "v2",
  "db": "reachable",
  "hardDisabled": false
}
```
